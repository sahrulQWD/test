Pertanyaan Yang sering di ajukan

Apa yang di maksud dengan website hosting?

Web Hosting adalah layanan penyimpanan data digital berupa teks, gambar atau video suatu website. Data website tersebut disimpan di server Hosting yang selalu aktif 24 jam dan terhubung oleh internet sehingga website dapat diakses secara online. Gunakan layanan web hosting terbaik yang menawarkan uptime 99.99% dan menggunakan data center Indonesia agar website dapat selalu diakses dengan performa website yang optimal.

Beli Hosting gratis Domain

Anda akan mendapat domain gratis jika Anda membeli paket Hosting Medium Performance atau High Performance dengan minimal kontrak 1 tahun

Apakah ada paket Hosting khusus WordPress?

Tentu saja. Qwords memiliki paket khusus WordPress yang bernama WordPress Hosting. Sebagai pengguna CMS WordPress sangat dianjurkan menggunakan paket WordPress Hosting ini karena spesifikasi resource pada hosting khusus WordPress sudah disesuaikan dengan keperluan website berbasis WordPress.

Kemampuan yang dimiliki oleh hosting ini lebih besar dibandingkan Shared Hosting biasa. Selain itu, paket WordPress Hosting dilengkapi dengan CloudPOP untuk kinerja WordPress yang optimal dan menggunakan teknologi SSD berkecepatan tinggi.

Apa yang dimaksud dengan Cloud?

Cloud merupakan teknologi penyimpanan data atau program yang terhubung dengan internet sehingga data dapat diakses kapanpun dan dimanapun tanpa harus membawa file berupa hard drive.

Paket Hosting apa yang terbaik buat saya?

Qwords menawarkan berbagai macam paket Hosting yang sudah disesuaikan resource nya dengan keperluan Anda. Jika Anda menggunakan website untuk blog pribadi, website sederhana untuk promosi produk dan email untuk penggunaan personal, Anda dapat menggunakan paket Value Performance.

Namun jika Anda menggunakan website untuk perusahaan, pemerintahan, bisnis online hingga e-commerce (toko online), paket Hosting Medium Performance dan High Performance dapat dijadikan pilihan. Jika Anda masih memiliki pertanyaan lebih, Anda dapat langsung live chat dengan Customer Service kami.

Apakah saya dapat migrasi Hosting ke Qwords?

Tidak perlu ragu untuk melakukan migrasi hosting ke Qwords karena proses nya mudah dan cepat. Silakan menghubungi Customer Service kami melalui call center atau live chat, Anda akan dibantu dan dijelaskan untuk tahapan migrasi hosting Anda.
